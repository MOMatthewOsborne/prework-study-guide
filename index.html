HTML

  • Html is used for layout
  • Elements in head and meta are not displayed, other than title
  • H1-H6 are used for headings

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can use class selectors by writing the class name with a . before
  • You can select multiple elements by separating them with ,

Git

  • Git status check which branch we are on
  • ls shows us what is in our current directory
  • pwd shows us what directory we are in
